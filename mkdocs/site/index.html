Accueil : TP2

@Auteur : TONDON César

Sujet

Ecrire une application JAVA qui simule le remplissage d’une baignoire qui fuit parallèlement. L'objectif est faire le bilan sur le "coût" de chaque simulation :

  • quantité d'eau utilisée
  • durée de remplissage
  • etc…

Précisions

L’application sera réalisée sous la forme d’une interface graphique interactive conçue avec Java FX. L’application devrait pouvoir être paramétrée facilement :

  • volume de la baignoire
  • débit du robinet de remplissage,
  • nombre de "trous" dans la baignoire
  • débit de fuite de chaque trou
